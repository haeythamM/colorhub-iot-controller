# ColorHub — Raspberry Pi Full-Stack Hardware UI

Browser-controlled hardware interface built with React, Flask, Raspberry Pi GPIO/PWM, I2C LCD, RGB LED, and buzzer feedback.

## Problem
Many IoT demos only display data. ColorHub focuses on reliable two-way control between a web UI and physical hardware.

## Features
- RGB LED control from browser UI
- 16x2 LCD status prompts
- Buzzer feedback
- Quiz mode with hardware responses
- Flask API on Raspberry Pi
- Structured state handling

## Tech Stack
React, TypeScript, Flask, Python, Raspberry Pi, GPIO/PWM, I2C, WebSocket/REST

## Architecture
Browser Dashboard → Flask API on Raspberry Pi → GPIO/PWM + I2C → RGB LED / LCD / Buzzer
